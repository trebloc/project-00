# Dot Racing

 The goal was to write a game using CSS to move the players by pixel for each step from left to right and then declare a winner.

## Technologies Used
* Javascript
* JQuery
* HTML
* CSS
* Bootstrap

## Existing Features
* Two Player Game
* Reset Button

## Planned Features
* Add Color
* Add Rounds
* Add a Track
