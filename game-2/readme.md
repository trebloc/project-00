# X's and O's

 The Goal was to write a game to move left to right using bootstrap boxes.  Also, added music of Ellie King's Ex's and Oh's since the game is is X's and O's. 

## Technologies Used
* Javascript
* JQuery
* HTML
* CSS
* Bootstrap
* Soundcloud Embed

## Existing Features
* Two Player Game
* Reset Button
* Music Soundtrack

## Planned Features
* Add Color
* Add Rounds
* Add a Track
